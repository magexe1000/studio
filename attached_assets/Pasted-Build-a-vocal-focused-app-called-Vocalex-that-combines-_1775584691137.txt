Build a vocal-focused app called "Vocalex" that combines vocal practice, pitch detection, vocal training, and recording into a single clean and optimized experience.

IMPORTANT:
- Keep UI minimal, clean, and consistent with the ecosystem (Chordex, Drumex, Mixex, Stagex)
- Focus on performance and real-time audio responsiveness
- App must work locally (no dependency on internet)

----------------------------------
1. CORE CONCEPT
----------------------------------

Vocalex is an all-in-one vocal tool that allows users to:

- Practice singing with tracks
- Monitor pitch in real time
- Train their voice
- Record and review takes

----------------------------------
2. MAIN SECTIONS
----------------------------------

Implement 4 core sections inside Vocalex:

1. Practice
2. Pitch
3. Vocal Lab
4. Takes

----------------------------------
3. PRACTICE (VOCAL PRACTICE MODE)
----------------------------------

- Allow user to sing along with audio tracks
- Integrate with multitrack system (like Mixex)

Features:

- Reduce/remove original vocals
- Control volume of tracks
- Microphone input monitoring
- Optional reverb for voice

Goal:
→ Let users practice singing over songs comfortably

----------------------------------
4. PITCH (REAL-TIME DETECTION)
----------------------------------

- Detect pitch from microphone input in real time

Display:

- Current note (e.g. A4, C#3)
- Visual indicator:
  - in tune
  - slightly off
  - out of tune

Behavior:

- Smooth and stable detection
- No jittery UI
- Fast response

----------------------------------
5. VOCAL LAB (TRAINING)
----------------------------------

Provide vocal exercises:

- Scales (ascending/descending)
- Warm-ups
- Breathing exercises

Features:

- Guided practice
- Visual pitch feedback
- Repeat exercises

Goal:
→ Improve vocal accuracy and control

----------------------------------
6. TAKES (RECORDING SYSTEM)
----------------------------------

- Allow recording of vocal sessions

Features:

- Record multiple takes
- Playback recordings
- Compare takes
- Rename or delete takes

Optional:
- Show basic waveform

----------------------------------
7. AUDIO ENGINE (CRITICAL)
----------------------------------

- Low latency microphone input
- Real-time processing
- Stable pitch detection
- No delay in monitoring voice

----------------------------------
8. PERFORMANCE
----------------------------------

- Must be smooth and responsive
- No lag in pitch detection
- Efficient audio handling

----------------------------------
9. LOCAL FUNCTIONALITY
----------------------------------

- Works offline
- Stores recordings locally
- No cloud dependency

----------------------------------
10. INTEGRATION READY
----------------------------------

- Compatible with Mixex (practice over multitracks)
- Scalable for future features

----------------------------------
11. FINAL RESULT
----------------------------------

Vocalex must:

- Feel like a professional vocal tool
- Be responsive and accurate
- Be simple to use but powerful
- Integrate well with the rest of the ecosystem

----------------------------------
GOAL
----------------------------------

Create a complete vocal app combining practice, pitch detection, training, and recording into one optimized and high-quality experience.
