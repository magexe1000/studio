Fix the system status bar visibility so that time, battery, and network indicators are always visible and properly styled across the entire app.

IMPORTANT:
- Do NOT break layout or UI design
- Ensure compatibility with Android (WebView / APK) and browser/PWA
- Must look clean and consistent with app theme

----------------------------------
1. ROOT CAUSE CHECK
----------------------------------

Check if the app is forcing fullscreen or hiding system UI:

- Remove or adjust any:
  - fullscreen flags
  - immersive mode
  - hidden status bar settings

----------------------------------
2. ANDROID (WEBVIEW / CAPACITOR)
----------------------------------

Ensure status bar is enabled:

- Do NOT use full immersive mode unless intentional
- Set status bar to visible

If using Capacitor:

- Configure status bar plugin properly
- Example:
  - visible: true
  - style: light or dark based on theme

----------------------------------
3. SAFE AREA HANDLING
----------------------------------

Ensure content does NOT overlap the status bar:

- Add top spacing using:
  - safe-area-inset-top
  - padding-top

Example:
- padding-top: env(safe-area-inset-top);

----------------------------------
4. PWA / BROWSER SUPPORT
----------------------------------

Check meta tags:

- Ensure proper viewport configuration
- Avoid forcing fullscreen unintentionally

----------------------------------
5. STATUS BAR STYLE
----------------------------------

Match status bar with app theme:

- Dark theme → light icons
- Light theme → dark icons

----------------------------------
6. PREVENT OVERLAP
----------------------------------

Ensure:

- No elements are covering the top area
- No absolute positioned elements blocking it

----------------------------------
7. TEST CASES
----------------------------------

Verify behavior:

- On app launch
- On navigation between sections
- In modals / live mode

----------------------------------
8. OPTIONAL (IMMERSIVE MODE CONTROL)
----------------------------------

If fullscreen is desired:

- Add toggle in Settings:
  - "Fullscreen Mode"

When OFF:
- Always show status bar

----------------------------------
GOAL
----------------------------------

Ensure the system status bar (time, battery, signal) is always visible, correctly styled, and never overlapped, providing a native and polished app experience.
